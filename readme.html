ArduVidRx Controller

The ArduVidRx Controller is an Android app for controlling an ArduVidRx receiver (an FPV video receiver that has been modified to have an Arduino board).  Communication is via Bluetooth.  The project is developed and built using Android Studio.

This project makes use of code from the BlueTerm and ApmeM FlowLayout projects.
